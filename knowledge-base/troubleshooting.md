# Troubleshooting Guide

Common issues and solutions for NovaSaaS.

## Login Issues

### "Invalid credentials" error

**Cause:** Incorrect email or password

**Solution:**
1. Verify your email address is correct
2. Reset your password using "Forgot Password"
3. Check for typos or extra spaces

### "Account locked" message

**Cause:** Too many failed login attempts

**Solution:**
- Wait 15 minutes and try again
- Contact support if issue persists

## Project Issues

### Cannot create projects

**Cause:** Permission or quota limits

**Solution:**
1. Check your plan limits in Settings > Billing
2. Verify you have Admin or Member role
3. Delete unused projects to free up quota

### Tasks not loading

**Cause:** Browser cache or network issues

**Solution:**
1. Refresh the page (Ctrl+F5)
2. Clear browser cache
3. Check internet connection
4. Try incognito/private mode

## Collaboration Issues

### Team member can't access project

**Cause:** Permission settings

**Solution:**
1. Go to Project Settings > Sharing
2. Verify team member has access
3. Check their role permissions

### Invitations not received

**Cause:** Email delivery issues

**Solution:**
1. Ask recipient to check spam folder
2. Verify email address is correct
3. Resend invitation from Team settings
4. Add noreply@novasaas.com to contacts

## Performance Issues

### Slow page loading

**Solution:**
1. Check internet connection speed
2. Disable browser extensions temporarily
3. Try a different browser
4. Clear browser cache and cookies

### App freezing

**Solution:**
1. Refresh the page
2. Check browser console for errors (F12)
3. Update browser to latest version
4. Reduce number of open tabs

## Billing Issues

### Payment failed

**Solution:**
1. Verify card details are correct
2. Check card expiration date
3. Contact your bank
4. Try a different payment method

### Can't upgrade plan

**Solution:**
1. Clear browser cache
2. Try incognito mode
3. Contact support for assistance

## Mobile Issues

### App crashes on startup

**Solution:**
1. Update app to latest version
2. Restart your device
3. Reinstall the app
4. Check device compatibility

### Sync not working

**Solution:**
1. Check internet connection
2. Pull to refresh manually
3. Log out and back in
4. Reinstall if issue persists

## Still Need Help?

If you can't find a solution here, our support team is ready to help:

1. Submit a ticket via the support form
2. Include detailed steps to reproduce
3. Attach screenshots if applicable
4. Note your browser and OS version
