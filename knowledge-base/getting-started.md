# Getting Started with NovaSaaS

Welcome to NovaSaaS! This guide will help you get up and running quickly.

## Quick Start

### 1. Create Your Account

Visit our signup page and create your account with your work email address.

### 2. Set Up Your Workspace

After signing in, you'll be prompted to create your first workspace:
- Choose a workspace name (e.g., "Acme Corp")
- Select your team size
- Choose your primary use case

### 3. Invite Team Members

Navigate to **Settings > Team** to invite collaborators:
1. Click "Invite Members"
2. Enter email addresses
3. Assign roles (Admin, Member, Viewer)
4. Send invitations

### 4. Create Your First Project

1. Click the "+" button in the sidebar
2. Select "New Project"
3. Enter project name and description
4. Choose a template or start from scratch

### 5. Add Tasks

Start breaking down your work:
- Click "Add Task" in your project
- Enter task details
- Assign to team members
- Set due dates and priorities
